# Using the ExtraChill AI Chat

Chat with our AI assistant to get help with music, events, and the Extra Chill community.

## Accessing the Chat

Visit [chat.extrachill.com](https://chat.extrachill.com) and log in with any Extra Chill account.

## How It Works

The AI chat uses advanced language models to provide helpful responses about:

- Music recommendations and discovery
- Event information and planning
- Community features and navigation
- General questions about Extra Chill

## Features

### Conversation History

- Your chat history is saved between sessions
- Continue conversations where you left off
- Clear history anytime to start fresh

### Tool Integration

The AI can use various tools to provide better answers:

- Search the Extra Chill network for content
- Access event calendars and artist information
- Provide up-to-date information from the web

### Multi-Turn Conversations

The AI can perform complex tasks by using multiple tools in sequence, such as searching for events and then providing directions.

## Tips for Better Results

- Be specific about what you're looking for
- Mention locations, artists, or genres when relevant
- Ask follow-up questions to refine results
- Use natural language - no need for special commands

## Privacy and Safety

- Conversations are private and associated with your account
- The AI follows community guidelines
- Sensitive information is handled securely

## Clearing Your History

To start fresh or remove old conversations:

1. Click the settings or menu button
2. Select "Clear History"
3. Confirm the action

## Need Help?

If you experience issues with the chat:

- Try refreshing the page
- Clear your browser cache
- Contact support at [extrachill.com/contact](https://extrachill.com/contact/)

The chat is continuously improved based on user feedback. Let us know how we can make it better!
